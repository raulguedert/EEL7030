# Roteiro 5

## Problema a ser resolvido

Escrever dados em um display do tipo LCD.

## Informações do Hardware

O display LCD de 16x2 (16 colunas e duas linhas) foi conectado ao ESP32-C3 da seguinte maneira:


RS --> GPIO8

R/W --> GND

ENABLE --> GPIO9


DATA0 --> GPIO0

DATA1 --> GPIO1

DATA2 --> GPIO2

DATA3 --> GPIO3

DATA4 --> GPIO4

DATA5 --> GPIO5

DATA6 --> GPIO6

DATA7 --> GPIO7
